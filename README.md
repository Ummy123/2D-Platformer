# 2D-Platformer
2D simple platfromer game

Project:

Author: Amer S. 
Date: January 30, 2022
About The Project: I wanted to create a simple game to demonstrate some basic concepts and have a fun game.

Prerequisites:
-python version 3.9
-pygame version 1.9.6

Controls:
a key - move left
d key - move right
spacebar - jump
